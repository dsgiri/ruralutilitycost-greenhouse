# Component Library Spec

**Priority:** High Priority
**Description:** Reusable UI components with props and variants

## Core Components
- `Shell`: Main application layout, navigation, and footer.
- `ToolCard`: Displays tool summary, category, and favorite toggle.
- `SEO`: Injects standard meta tags, Open Graph, and JSON-LD schema.
- `CookieBanner`: Manages user consent for tracking.
