# User Flows & Journey Maps

**Priority:** High Priority
**Description:** Step-by-step journeys for each user persona

## Flow: Estimate Startup Cost
1. User navigates to Home Dashboard.
2. Clicks on "Greenhouse Startup Cost Calculator".
3. Enters structure type, growing area, and automation level.
4. Reviews CapEx breakdown and per-square-foot cost.

## Flow: Save Favorite Tool
1. User clicks the heart icon on any tool card.
2. Tool is added to `localStorage`.
3. User navigates to `/favorites` to quickly access the tool in future sessions.
