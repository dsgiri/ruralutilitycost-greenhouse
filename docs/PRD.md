# Product Requirements Document (PRD)

**Priority:** Critical #01
**Description:** Product vision, user personas, and feature list

## Product Vision
Build a greenhouse economics and operations hub for controlled-environment agriculture to help growers estimate costs and profitability.

## User Personas
- Greenhouse growers
- Vegetable producers
- Ag investors
- CEA planners

## Feature List
- Greenhouse Cost Estimator
- Heating and Energy Calculator
- ROI and Payback Tool
- Crop Profitability Tool
- Scenario Comparison
- Favorites
